# HH Frontend — Поиск вакансий Frontend-разработчика

## Установка

### Установка зависимостей

npm install

### Запуск в режиме разработки

npm run dev

### Сборка проекта

npm run build

### Запуск тестов

npm run test

### Линтинг

npm run lint

## Сборка

### Создать production сборку

npm run build

### Предпросмотр production сборки

npm run preview

Результат сборки находится в папке `dist/`.

## Деплой

Проект развёрнут на GitHub Pages:

### Деплой на GitHub Pages

npm run deploy
